# 💬 Flask Live Chat App

A real-time WhatsApp-like chat application built with **Flask**, **Flask-SocketIO**, and **MySQL**.

![Chat Preview](https://via.placeholder.com/800x400/00cc77/ffffff?text=Live+Chat+App)

## ✨ Features

- ✅ User Registration & Login
- ✅ Add Contacts with Custom Display Names
- ✅ Private Real-time Chat (1-to-1)
- ✅ Online/Offline Status (Green Dot)
- ✅ Indian Time (IST) with AM/PM format
- ✅ Emoji Picker
- ✅ Image/File Sharing
- ✅ Message History (Saved in Database)
- ✅ Clean & Modern Dark UI

## 🛠️ Tech Stack

- **Backend**: Flask + Flask-SocketIO
- **Database**: MySQL (XAMPP)
- **Frontend**: HTML, CSS, JavaScript + Socket.IO
- **Real-time**: WebSockets
